# Прогнозирование дохода клиента банка

В репозитории находятся два основных варианта решения задачи:

- `final.ipynb` — лучшее итоговое решение;
- `best.ipynb` — второй по качеству вариант решения.

Для корректного запуска все ноутбуки, исходные данные, JSON-файлы с параметрами и файл зависимостей должны находиться в одной папке.

## 1. Склонировать репозиторий

git clone https://github.com/imydon/alpha_hack_2026.git


## 2. Установка библиотек

Обновите `pip` и установите зависимости:

```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Для работы с Jupyter Notebook дополнительно установите Jupyter, если его нет в `requirements.txt`:

```bash
python -m pip install notebook ipykernel
```

## 3. Проверка файлов

До запуска убедитесь, что в одной папке находятся:

- `train.csv`;
- `test.csv`;
- `sample_submission.csv`, если он используется при формировании результата;
- все необходимые JSON-файлы с гиперпараметрами;
- `final.ipynb` и `xgb_best.ipynb`;
- `requirements.txt`.

Не перемещайте файлы после запуска ноутбука. Относительные пути в коде рассчитаны на то, что все файлы лежат рядом.

## 5. Запуск Jupyter Notebook

1. `final.ipynb` — для воспроизведения лучшего результата;
2. `best.ipynb` — для воспроизведения второго результата.
